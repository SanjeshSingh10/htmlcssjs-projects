# GitHub Developer Dashboard

A simple web application that allows users to search for a GitHub profile using the GitHub REST API.

## Current Features

- Search GitHub users by username
- Display profile information
  - Avatar
  - Name
  - Username
  - Bio
  - Location
  - Public Repositories
  - Followers
  - Following
- View GitHub Profile
- Input validation
- Error handling for invalid usernames

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub REST API
